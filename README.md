# Live Search Filter

A simple live search filter built with HTML, CSS, and Vanilla JavaScript.

This project allows users to search through a list of fruits in real time while highlighting the matching text dynamically.

## Features

- Instant live search
- Dynamic filtering with JavaScript
- Highlight matching text
- "No result" message
- Responsive design
- Clean and simple UI

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## How It Works

- The app listens to user input events.
- The fruit list is filtered dynamically using `filter()`.
- Matching text is highlighted using string slicing and `innerHTML`.
